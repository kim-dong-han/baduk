# CLAUDE.md

이 저장소 작업 시 따른다. 목적: Claude 사용량 최소화.

## 세션 시작 절차
1. **PROJECT.md** 읽기 (소개·스택·실행).
2. 위치/구조 필요 시에만 **ARCHITECTURE.md** (코드 재탐색 대체).
3. 작업 규칙은 **RULES.md** — 특히 *변경분만 출력, 설명 3줄 이내*.
4. 상태는 **TODO.md**, 요청 템플릿은 **PROMPTS.md**.
- 위 문서에 있는 사실은 다시 코드로 확인하지 말 것(바뀐 정황 있을 때만 검증).

## 절대 규칙
- 변경된 부분만 출력. 전체 파일 재출력 금지.
- 설명 최소. 무관한 파일 읽지 않기(컨텍스트 오염 방지).
- 구조 변경 → ARCHITECTURE.md / 상태 변경 → TODO.md 동시 갱신.
- 상세는 RULES.md를 따른다.
